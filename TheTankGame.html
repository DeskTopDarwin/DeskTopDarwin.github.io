The Tank Game

A game with a tank as main mechanic. The goal of the game is to exhaust the enemy points while maintaning your's. To do so, you will need to, assisted by Ai soldier teammates, capture points. The enemy forces arsenal will be composed of rifles and rocket launchers.

Gameplay

More on the project

The game was built solo in Unity while using assets of the assets store. To keep a structured environment, I used Agile scrum methodology to keep track of the progress done and what I had to complete. The biggest challange was making an AI simple enough to have a lot of units (well over a hundred if needed) and still run smooth. The AI uses nav mesh to move around while using a state machine as the brain and uses raycasts to aid in the targeting. The game was built with component based architecture therefore the interactions between scripts were kept simple.
